# California-Housing-Prediction
مشروع التنبؤ بأسعار المنازل في كاليفورنيا

يعد هذا العمل الخطوة الأولى في دراسة تطبيقات تعلم الآلة، حيث يركز على استخدام خوارزمية الانحدار الخطي للتنبؤ بالقيم السعرية بناءً على البيانات الرسمية لولاية كاليفورنيا. الهدف الأساسي من المشروع هو فهم آليات تدريب النماذج الرياضية وتقييم دقتها بشكل علمي.

من الناحية التقنية، تم تطبيق خوارزمية Linear Regression مع تقسيم البيانات إلى مجموعتي تدريب واختبار لضمان موضوعية التقييم. تم قياس الأداء باستخدام معايير MSE و R2 Score كأدوات أساسية لقياس جودة التوقعات الناتجة عن النموذج.

اعتمد المشروع على لغة Python ومكتبات Scikit-learn و NumPy و Matplotlib. تم تنظيم العمل برمجياً باستخدام بيئة افتراضية (Virtual Environment) مع توثيق كافة الاعتمادات في ملف requirements.txt لضمان استقرار البيئة البرمجية عند إعادة التشغيل.

أظهرت النتائج قدرة النموذج على رصد الأنماط العامة للبيانات، وهو ما يظهر بوضوح في الرسم البياني المرفق الذي يقارن بين القيم الحقيقية والقيم التي تنبأ بها النموذج.

لتشغيل المشروع، يتم تثبيت المكتبات المطلوبة عبر الأمر:
'pip install -r requirements.txt'
ثم تنفيذ الملف الأساسي:
python housing_prediction_v1.py
